# Shashank Kaneria - Personal Website

A beautiful, modern, and dynamic personal website showcasing Shashank Kaneria's bio, development work, blog, and photography.

## Structure

- `index.html`: Home page with grey/white theme and colored navigation boxes
- `bio.html`: About me section with resume
- `development.html`: Showcase of development projects
- `blog.html`: Blog posts and articles
- `photography.html`: Coming soon page for photography portfolio

## Usage

1. Open `index.html` in your web browser to view the home page.
2. Click on the colored boxes to navigate to different sections.
3. Each section has a navigation link back to the home page.

## Customization

- **Photo**: Replace placeholder images with your actual photos in each HTML file
- **Content**: Edit the content in each section's HTML file
- **Colors**: Modify the CSS for different color schemes
- **Resume**: Update the resume section in `bio.html` with your actual information
- **Projects**: Add real project details in `development.html`
- **Blog Posts**: Expand blog content in `blog.html`
- **Photography**: Replace the coming soon message with actual portfolio in `photography.html`

## Features

- Responsive design
- Smooth animations and hover effects
- Clean grey/white theme on home page
- Color-coded sections with white text
- Separate pages for each content area

## Technologies Used

- HTML5
- CSS3 (with modern features like flexbox, transitions, and box-shadow)
- Vanilla JavaScript for animations
